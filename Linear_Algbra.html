LINEAR ALGEBRA

Course Objective

This course will introduce the basic elements of linear algebra with a focus towards solving linear equations and matrix algebra. These concepts have several applications in computer science, e.g., graphs and networks, Markov matrices, computer graphics, cryptography, etc.

Syllabus

Vectors and matrices; linear equations; elimination; vector spaces; orthogonality; determinants; eigenvalues and eigenvectors; singular value decomposition; complex matrices; Code Vectorization using matrices, Linear Algebra in Data Science

Topics we have covered

  • Vector addition
  • Scalar and Vector Product
  • Matrix multiplication
  • Adjoint, Transpose, Inverse
  • Methods to find solution of Linear equations
  • Methods to find inverse of matrix
  • Vector Space and Subspace
  • Null Space and Column Space
  • Basis and Dimensions
  • Rank of a matrix
